## 1.0.0
* V81 compatibility!
* Added functionality for Cadaver Blooms, blooming, and Feiopars.
* Major code refactor to make things look nicer.
* Explosion resistant enemies (such as the Maneater) no longer spawn multiple explosions, but will still die the same way.

## 0.3.0

* Made config settings synced to the host (CSync is now a hard dependency). 
* Fixed a bug with Circuit Bees not exploding when a player is at or below 10 health.

## 0.2.0

* Added a config setting to turn off enemy explosions. 
* Fixed a config bug with `patchGiantKiwi` always being set to true.
